THE FULL ROSTER

This page is ordered like a UI progression timeline: the earliest and rougher interface work starts at the top, while the stronger, more polished UI projects sit closer to the bottom.

Unity C# 3D Team Project SAE Comic Con 2024

IT'S JUST A RAGE GAME

A rage-platformer project I worked on with a friend. It started as a chaotic challenge game and grew enough to be shown at SAE's Comic Con display in 2024. The UI is intentionally kept early in the roster because this page is ordered from my rougher UI work toward my strongest interface work.

Unity C# Golf Game Atmospheric Music

ETHEREAL GOLF

A calm, aesthetic golf game built around simple aiming, relaxing visuals, and pleasing background music. No K-pop in this one yet — just a clean, mood-driven golf experience where the atmosphere carries the feel of the game.

Unity C# 2D / Casual Timer System Gameplay Loop

SLEEPER'S JOURNEY

A small survival-style sleep protection game where the goal is to keep the person on screen asleep for a set amount of time. The player has to kill flies, use bug spray to keep bees away, and manage every disturbance before the sleeper wakes up.

Unity C# Racing Button Mash Local Play

BLOCKY DERBY

A block-racing game inspired by the energy of Uma Musume-style races. You play as a block competing against other blocks, aggressively tapping the button to push your racer forward and fight for first place.

Unity C# Couch Co-op Low Poly Sports

WE PLAYING BASKETBALL

A simple couch co-op basketball game where players control LeBron-styled low-poly characters. The goal is straightforward: get the ball in the hoop and outscore whoever is sitting next to you.

Unity C# Mirror Network Local Co-op Platform Fighter

BALL BUSTERS

A local co-op arena fighting game built on the Mirror networking system. Players control circular characters known as balls, each with a different feel, and battle in a boxing ring by aggressively bumping opponents out of the arena. Think Super Smash Bros energy, but with balls, impact, and chaos. This is my second-best UI project so far.

HTML5 CSS3 JavaScript Canvas API Web Game Itch.io Best UI

COSMO CLASH GRAND GRAVITY

A K-pop inspired survival card game built entirely in HTML, CSS, and JavaScript. Players build a subunit by selecting members across themed mission rounds including dance, vocal, rap, and concept challenges. It includes idol cards, an AI opponent, custom CSS animations, audio, and a Canvas API arena battle view.

This is one of my strongest UI-focused web games so far, so it sits after Ball Busters in the portfolio progression. The buttons below let visitors either play the hosted itch.io version or open the project slideshow to view the screenshots and files.
